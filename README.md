GitHub User Finder (Поиск пользователей GitHub)

Абдульманов Руслан Рустемович

Описание
GUI-приложение на Python, которое позволяет искать пользователей GitHub по логину или имени, просматривать результаты в табличном виде, добавлять избранных пользователей и сохранять их в локальный JSON-файл.

Основные возможности
- Поиск пользователей через официальный GitHub Search API.
- Отображение логина, URL аватара и ссылки на профиль.
- Просмотр детальной информации о пользователе (имя, компания, количество репозиториев и т.д.).
- Добавление/удаление пользователей в/из избранного.
- Автоматическое сохранение избранного в файл favorites.json.
- Проверка на пустой ввод перед поиском.

Требования и установка
1. Python 3.6 или выше.
2. Установите необходимые библиотеки:
   pip install requests
   Библиотека requests обязательна – она используется для отправки HTTP-запросов к GitHub API.
   Для отображения аватаров в интерфейсе (опционально) можно установить Pillow:
   pip install pillow
   (Текущая версия не выводит аватары, но их легко добавить при необходимости.)

Как запустить приложение
1. Скачайте файл github_user_finder.py.
2. Запустите его командой:
   python github_user_finder.py
3. Введите имя или часть логина GitHub в верхнее поле и нажмите «Поиск» (или Enter).
4. Выберите пользователя в таблице и воспользуйтесь кнопками «Добавить в избранное» или «Показать детали».
5. Избранное отображается в нижней панели, откуда его можно удалить.

Как использовать API
- Приложение использует REST API GitHub, которое не требует аутентификации для базового поиска (ограничение: 60 запросов в час с одного IP).
- При необходимости можно повысить лимит, добавив персональный токен в заголовки (раскомментировав и изменив строку в коде):
  headers = {"Authorization": "token ваш_токен"}
- Используемые эндпоинты:
  - GET /search/users?q={запрос} — поиск пользователей.
  - GET /users/{логин} — детальная информация о пользователе.

Примеры использования
1. Поиск пользователя:
   - Введите "torvalds" → нажмите «Поиск». В таблице появится Linus Torvalds.
   - Выделите строку → «Показать детали» покажет компанию (Linux Foundation) и количество репозиториев.
2. Работа с избранным:
   - Добавьте несколько пользователей в избранное. При повторном запуске они загрузятся из favorites.json.
3. Обработка ошибок:
   - Попытка поиска с пустой строкой → появится предупреждение.
   - При превышении лимита API выводится сообщение об ошибке с HTTP-статусом.
